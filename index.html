Magamról

Évek óta foglalkozom webfejlesztéssel. Eleinte saját magamtól, az internetet segítségül hívva kezdtem tanulmányozni a HTML és CSS nyelveket. Statikus weeblapokat készítettem saját szórakoztatásomra.

Később, hogy kis dinamikát is vigyek az oldalakba az űrlapokba bevitt adatok tárolása kedvééret, a PHP felé fordult érdeklődésem. Szerver oldalon a PHP mellett megismerkedtem a SQL alapjaival, így már képes voltam adatokat tárolni és lekérdezni.

Tudásom teljessé csak a Javascript megismerése után vált. Ezzel az ismerettel már sokkal élőbbé tudtam varázsolni a weboldalakat.
